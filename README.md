# LOL.VS 프로토타입

리그 오브 레전드 플레이 영상을 등록하고, 근거 있는 의견과 다음 플레이 피드백을 받는 서비스 프로토타입입니다.

## 프로토타입에서 가능한 것

- 닉네임과 숫자 4자리로 가입·로그인하는 프로토타입 계정
- 일반 사용자의 플레이 판정·플레이 피드백 영상 등록
- 최대 10개 영상 접수 제한과 관리자 검토 후 공개
- 실제 로컬 영상 선택, 브라우저 저장, 재생
- 로그인 사용자 댓글 작성
- 관리자 공식 판정·전문 피드백 등록
- 일반 사용자의 판정·피드백 권한 요청 안내
- 더미 콘텐츠 없는 빈 게시판과 랭킹

Riot API, Supabase, Mux 백엔드는 아직 연결되지 않았습니다. 로그인 정보와 업로드 영상은 현재 사용하는 브라우저에만 저장됩니다. 여러 사용자가 영상을 공유하려면 Supabase와 Mux 프로젝트 연결이 필요합니다.

## 실행

```bash
npm install
npm run dev
```
