# LOL.VS 프로토타입

티어가 인증된 리그 오브 레전드 플레이어들이 억울한 플레이 영상을 보고, 누구의 잘못인지 투표하고 개선 피드백을 남기는 커뮤니티 프로토타입입니다.

## 프로토타입에서 가능한 것

- 사건 목록과 인기 재판 둘러보기
- 사건 상세 화면 열기
- A/B 잘못 투표 후 결과 확인
- 플레이 피드백 댓글 작성
- 영상 사건 제보 흐름 체험
- 인증 티어와 사이트 활동명 분리 확인

실제 Riot 로그인, 영상 저장, 데이터베이스 기능은 아직 연결되지 않은 UI 프로토타입입니다.

## 실행

```bash
npm install
npm run dev
```

## GitHub Pages 빌드

```bash
NEXT_PUBLIC_BASE_PATH=/lol.vs npm run build:pages
```
