# LOL.VS 프로토타입

티어가 인증된 리그 오브 레전드 플레이어들이 억울한 플레이 영상을 보고, 누구의 잘못인지 투표하고 개선 피드백을 남기는 커뮤니티 프로토타입입니다.

## 프로토타입에서 가능한 것

- 사건 목록과 인기 재판 둘러보기
- 사건 상세 화면 열기
- 활동 닉네임과 티어를 선택하는 데모 로그인
- 로그인 사용자만 A/B 투표, 댓글, 대댓글, 신고 참여
- 선택한 판결과 타임스탬프 근거가 표시되는 피드백
- 실제 로컬 영상 선택, 브라우저 저장, 재생
- 커뮤니티 글 작성과 카테고리 게시판
- 판결 정확도·도움 피드백 기반 랭킹
- 사이트 이용 가이드와 FAQ

Riot API와 서버 백엔드는 아직 연결되지 않았습니다. 로그인 정보와 업로드 영상은 현재 사용하는 브라우저에만 저장되며, 실제 티어 인증이나 다른 사용자와의 영상 공유는 지원하지 않습니다.

## 실행

```bash
npm install
npm run dev
```
